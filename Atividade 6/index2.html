Página 2 - Elementos Semânticos Gerais

article Define conteúdo independente e autocontido.
aside Define o conteúdo, além do conteúdo da página.
details Define detalhes adicionais que o usuário pode visualizar ou ocultar.
figcaption Definir uma legenda para um elemento figure.
figure Especifica conteúdo independente, como ilustrações, diagramas, fotos, listagens de códigos, etc.
footer O footer elemento define um rodapé para um documento ou seção.
header Especifica um cabeçalho para um documento ou seção.
main Especifica o conteúdo principal de um documento.
mark Define o texto marcado/destacado.
nav O nav elemento define um conjunto de links de navegação.
section Define uma seção em um documento.
summary Define um cabeçalho visível para um elemento details.
time Define uma data/hora.
